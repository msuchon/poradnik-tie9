Poradnik GitHub


GitHub jest platformą, działającą na zasadzie hostingu, przeznaczoną na projekty progamistyczne. Dzięki temu serwisowi możesz za darmo podzielić się z innymi tym, nad czym aktualnie pracujesz tak, aby inni mogli z tego skorzystać bądź współtworzyć z Tobą! Portal stworzony został przez Chrisa Wanstrath'a, PJ Hyett oraz Toma Prestona-Wernera i działa od kwietnia 2008 roku. W 2018 serwis przejęty został przez firmę Microsoft. GitHub opiera się o system kontroli wersji o nazwie Git.
